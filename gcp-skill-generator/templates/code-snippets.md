# Code Snippets 模板（仅 `cli_applicability: sdk-only` 适用）

> **Purpose:** 为 CLI 不足以覆盖完整功能、必须依赖 SDK（API 方式）的 skill，生成
> 可直接 `go run` 的标准化 Go SDK 代码片段。每个 Operation 一个独立 `.go` 文件，附带
> 独立 `go.mod` + `README.md`，便于 Agent 或用户在沙箱里独立编译运行。
>
> **适用条件：** `cli_applicability: sdk-only` （CLI 完全不暴露该 product/operation）
>
> **生成时机：** [Evaluation-Driven Generation Workflow §Step 5.5](../SKILL.md#step-55-generate-code-snippets-when-sdk-only)

---

## 1. 目录结构

```
gcp-[product]-ops/
└── assets/
    └── code-snippets/                # 仅 sdk-only 适用，dual-path/cli-first 不生成
        ├── go.mod                    # 独立 module, 路径示例: gcp-skills/gcp-[product]-ops/assets/code-snippets
        ├── go.sum                    # go mod tidy 生成
        ├── README.md                 # 使用说明 + 凭据环境变量 + 运行步骤
        ├── main.go                   # 公共 client 初始化 (可被各 snippet 引用)
        ├── create_<resource>.go      # 每个 operation 一个 .go 文件
        ├── describe_<resource>.go
        ├── modify_<resource>.go
        ├── delete_<resource>.go
        └── ...                       # 其它 product-specific operation
```

**文件命名规范：**
- `<verb>_<resource>.go`（如 `create_instance.go`、`describe_databases.go`）
- 全部小写、下划线分隔
- 动词需与 REST API 动词对齐（Create / Describe / Modify / Delete / List / Get / Start / Stop / Restart 等）

## 5. README.md 模板

```markdown
# Code Snippets — gcp-[product]-ops

## Prerequisites

- Go 1.21+
- Service account key: `export GOOGLE_APPLICATION_CREDENTIALS=/path/to/key.json`
- Project: `export CLOUDSDK_CORE_PROJECT=my-project`

## Available Snippets

| File | Description | Key Output |
|------|-------------|------------|
| `create_<resource>.go` | Create a new [resource] | $.name |
| `list_<resources>.go` | List all [resources] | $.items[].name |
| `delete_<resource>.go` | Delete a [resource] | $.operationType |

## Run

```bash
export GOOGLE_APPLICATION_CREDENTIALS=/path/to/key.json
export CLOUDSDK_CORE_PROJECT=my-project

go run .
```

## Security

- **NEVER** hardcode service account keys in these files
- **NEVER** print credential values
- Use environment variables for all configuration
